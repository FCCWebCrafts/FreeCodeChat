How to chat

Available Commands:

  • /list - get a list of users in your current room
  • /users - get the number of users in your current room
  • /listall - get a global list of users connected to the server
  • /usersall - get the global number of users connected to the server
  • /room - get the name of your current room
  • /roomall - get a global list of rooms
